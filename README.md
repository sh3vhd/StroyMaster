# 🏗️ СтройМастер — Сайт строительной компании

Современный одностраничный сайт для строительной компании полного цикла. Создан на React + TypeScript + Vite + Tailwind CSS с собственным набором SVG-иконок (без внешних иконочных библиотек).

## ✨ Возможности

- 🎨 **Современный дизайн** с использованием Tailwind CSS 4
- 📱 **Полностью адаптивный** интерфейс (mobile-first)
- ⚡ **Быстрая загрузка** благодаря Vite и оптимизации
- 🧩 **Собственные SVG-иконки** — ни одной внешней зависимости для иконок
- 🎯 **Готовая структура секций** для строительного бизнеса
- 🌊 **Плавные анимации** и микро-взаимодействия
- 📝 **Форма обратной связи** с валидацией
- ♿ **Доступность** — семантическая HTML-разметка

## 🗂️ Секции сайта

| Секция | Описание |
|--------|----------|
| **Hero** | Главный экран с УТП, статистикой и CTA |

## 🛠️ Технологии

- **React 18** + **TypeScript** — UI и типизация
- **Vite** — сборка и dev-сервер
- **Tailwind CSS 4** — стилизация
- **SVG** — все иконки и иллюстрации проектов нарисованы вручную

## 🎨 SVG-иконки

Все иконки находятся в `src/components/Icons.tsx`. Каждая иконка — это типизированный React-компонент:

```tsx
import { HouseIcon, HammerIcon, ShieldCheckIcon } from './components/Icons';

<HouseIcon size={24} className="text-orange-500" />
```

## 🚀 Запуск проекта

### Установка зависимостей

```bash
npm install
```

### Режим разработки

```bash
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173).

### Production-сборка

```bash
npm run build
```

Готовые файлы появятся в папке `dist/`.

### Предпросмотр сборки

```bash
npm run preview
```

## ⚙️ Кастомизация

### Изменить контакты

Контакты прописаны в компонентах `Navbar.tsx`, `Contact.tsx`, `Footer.tsx`.
Найдите строки с `+7 (495) 123-45-67`, `info@stroymaster.ru` и адресом — замените на свои.

### Добавить услугу

Откройте `src/components/Services.tsx` и добавьте элемент в массив `services`:

```tsx
{
  icon: HouseIcon,
  title: 'Новая услуга',
  description: 'Описание услуги',
  price: 'от 10 000 ₽/м²',
}
```

### Добавить проект

В `src/components/Projects.tsx` добавьте новый объект в массив `projects` со своей SVG-иллюстрацией.

### Поменять цветовую схему

Используйте поиск-и-замену в коде:
- `orange-500` → ваш основной цвет
- `orange-600` → ваш hover-цвет
- `orange-100` → ваш светлый фон
